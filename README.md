# pbtemplates

## Synopsis

Personnal Bash templates

## File hierarchy

```
./
├── cli/
│   ├── cli-main
│   └── cli-min
├── readme/
│   └── README.md
├── setup/
│   ├── setup-full
│   ├── setup-main
│   └── setup-min
└── ui/
    └── ui.bash
```

  - **cli:** applications (`./cli [-h | -v]`)
  - **readme:** README files
  - **setup:** installation scripts (`./setup install`)
  - **ui:** user interface functions
